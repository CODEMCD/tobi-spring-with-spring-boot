# 토비의 스프링 3.1 with spring boot
토비의 스프링 3.1 책의 예제를 Spring Boot와 같은 현재 사용하고 있는 환경에 맞게 작성해본다.

## 환경
- Spring Boot 2.3.3
- Java8
- H2 Database (not memory)
- JUnit5
- Lombok
